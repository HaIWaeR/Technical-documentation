1. **Найдите кнопку "Code"**  
   На главной странице репозитория найдите зелёную кнопку "Code" в правом верхнем углу.

2. **Нажмите на кнопку**  
   Нажмите на кнопку "Code" - откроется выпадающее меню с настройками клонирования.

3. **Выберите вкладку "HTTPS"**  
   В открывшемся меню, в разделе "Clone", убедитесь что выбрана вкладка **"HTTPS"** (она будет выделена).

4. **Скопируйте ссылку**  
   В поле ниже отображается ссылка на ваш репозиторий. Нажмите на иконку копирования или выделите и скопируйте ссылку вручную:

#### Скачивает все файлы проекта на ваш компьютер
```bash
git clone https://github.com/HaIWaeR/Technical-documentation.git
```

### Коммит проекта на git 

#### Подготавливает ВСЕ изменённые файлы для сохранения
```bash
git add .
```

#### Сохраняет изменения локально и отправляет коммит на git
```bash
git commit -m "Ваше сообщение"
```

#### Загружает изменения на GitHub
```bash
git push
```
