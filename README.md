### Три основные команды Git для работы

#### Команда 1: Клонирование (скачивание) репозитория
```bash
git clone https://github.com/HaIWaeR/Technical-documentation.git
```
**Что делает:** Скачивает все файлы проекта на ваш компьютер


#### Команда 2: Добавление изменений в коммит
```bash
git add .
```
**Что делает:** Подготавливает ВСЕ изменённые файлы для сохранения
*Вариант для одного файла:*
```bash
git add README.md
```


#### Команда 3: Создание и отправка коммита
```bash
git commit -m "Ваше сообщение"
git push origin main
```
**Что делает:**
- `git commit` — сохраняет изменения локально
- `git push` — загружает изменения на GitHub

**Важно:** Исходный файл KT_1 сохранён и не удалён!
