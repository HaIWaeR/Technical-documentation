# Техническая документация - ТИШИНА

## 1. ОБЩЕЕ ОПИСАНИЕ

Жанр: Локальный кооперативный хоррор
Платформа: PC
Количество игроков: 2
Режим: Split-screen кооператив

## 2. ТРЕБОВАНИЯ К ГЕЙМПЛЕЮ

### 2.1. Базовые механики

Управление двумя персонажами с разделенными способностями
Система распространения звука в трехмерном пространстве
Физика взаимодействия с окружающей средой
Система инвентаря и ресурсов

### 2.2. Система звука

Требования к звуковому движку:
- Реализация трехмерного позиционирования звука
- Система эмиттеров звука для игровых действий
- Расчет распространения звуковых волн
- Визуализация звуковых волн для игроков

Уровни громкости действий:
- Тихая ходьба - минимальный радиус
- Обычная ходьба - средний радиус
- Бег - большой радиус
- Взаимодействие с механизмами - максимальный радиус

### 2.3. Искусственный интеллект

Поведение врагов (Эхо):
- Патрулирование по заданным маршрутам
- Реакция на звуковые стимулы
- Поиск источника звука при обнаружении
- Прекращение поиска при отсутствии новых стимулов

Состояния AI:
- Состояние покоя (патрулирование)
- Состояние настороженности
- Состояние преследования
- Состояние поиска

## 3. СИСТЕМА ПЕРСОНАЖЕЙ

### 3.1. Следопыт

Способности:
- Использование эхолокатора для обнаружения врагов
- Пониженный уровень шума при движении
- Возможность видеть патрульные пути врагов

Ограничения:
- Не может взаимодействовать с сложными механизмами
- Не имеет доступа к системе крафта

### 3.2. Инженер

Способности:
- Взаимодействие с механизмами и терминалами
- Доступ к системе крафта
- Использование шумомера для оценки уровня шума

Ограничения:
- Создает больше шума при перемещении
- Не может использовать эхолокатор

## 4. СИСТЕМА РЕСУРСОВ И КРАФТА

### 4.1. Типы ресурсов

- Батарейки (источник энергии)
- Электронные компоненты (крафт)
- Медикаменты (восстановление здоровья)
- Детали механизмов (активация устройств)

### 4.2. Крафт предметов

Доступные предметы:
- Шумовые гранаты
- Аптечки
- Временные источники света
- Устройства звуковой маскировки

## 5. ДИЗАЙН УРОВНЕЙ

### 5.1. Структура уровней

- Нелинейное построение пространства
- Зоны различной освещенности
- Обязательные кооперативные преграды
- Скрытые области для исследования

### 5.2. Освещение

- Динамическая система освещения
- Источники света с ограниченным временем работы
- Зоны полной темноты
- Аварийное освещение

## 6. ИНТЕРФЕЙС ПОЛЬЗОВАТЕЛЯ

### 6.1. Элементы HUD

- Индикатор здоровья
- Уровень заряда батареи фонаря
- Индикатор создаваемого шума
- Состояние эхолокатора/шумомера

### 6.2. Split-screen реализация

- Равное разделение экрана
- Независимые камеры для каждого игрока
- Синхронизация важной информации
- Миникарта для навигации

## 7. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### 7.1. Графика

- Поддержка динамического освещения
- Система частиц для эффектов звука
- Шейдеры для визуализации эхолокации
- Пост-обработка для создания атмосферы

## 8. СОХРАНЕНИЯ

### 8.1. Система сохранений

- Контрольные точки прогресса
- Сохранение состояния инвентаря
- Запись найденных аудиозаписей

## Система звука - уровни громкости

| Действие | Радиус звука | Описание |
|----------|-------------|----------|
| Тихая ходьба | Минимальный | Для скрытного перемещения |
| Обычная ходьба | Средний | Стандартное перемещение |
| Бег | Большой | Быстрое, но шумное перемещение |
| Взаимодействие с механизмами | Максимальный | Длительные действия |

## Сравнение персонажей

| Параметр | Следопыт | Инженер |
|----------|----------|---------|
| **Основная способность** | Эхолокатор | Работа с механизмами |
| **Вторичная способность** | Тихий шаг | Шумомер |
| **Особенность** | Видит пути врагов | Доступ к крафту |
| **Ограничение** | Не работает с механизмами | Шумный |
| **Другое ограничение** | Нет доступа к крафту | Нет эхолокатора |
